# PaymentGatewayApi

## Running in Local

Pre-requisites:
* MongoDB
* MongoDB Compass
* .NET 5.0

Steps (.NET CLI):
* `cd PaymentGatewayApi`
* `dotnet run`

Steps (Visual Studio):
* Change launch profile to 'PaymentGatewayApi' (not IIS)
* Run the application

## Running in Docker

Pre-requisites:
* Docker-Desktop

Steps:
* `docker-compose up -d --build`
